### 总和运算子

> **总和运算子**(summation operator)是用以表示多个数的总和运算的一个有用的缩写符号，它在统计学和计量经济学分析中扮演着重要的角色。如果$$ \{ x_i : i = 1, \cdots ,n\} $$表示n个数的一个序列，那么我们就把这n个数的总和写为：
> 

$$\sum_{i=1}^n x_i = x_1 + x_2 + \cdots + x_n $$ 

**关键性质:**

- 对任意常数c有：$$ \sum_{i=1}^n c = nc $$
- 对任意常数c有：$$ \sum_{i=1}^n cx_i = c\sum_{i=1}^n x_i  $$
- 如果$$ \{ (x_i , y_i ) : i = 1,2,\cdots,n \} $$ 是n双数的一个集合，并可a和b是常数，那么

$$ \sum_{i=1}^n (ax_i + by_i) = a\sum_{i=1}^n x_i + b\sum_{i=1}^n y_i $$

**常见错误认识：**

- 比率的总和不等于总和的比率
- 平方的总和不等于总和的平方

### 描述统计

#### 均值

> 给定n个数$$ \{x_i :i = 1,\cdots,n\} $$ ，把它们加起来再除以x，便算出它们的平均数或均值：
> 
> $$ x = \frac{1}{n} \sum_{i=1}^n x_i$$
> 
> 这个统计量描述了点$$ x_i $$的集合趋势。

**基本性质**

- 离差之和必为零(d表示离差)：

$$ \sum_{i=1}^n d_i = \sum_{i=1}^n (x_i -  \bar x)  = \sum_{i=1}^n x_i - \sum_{i=1}^n \bar x = \sum_{i=1}^n x_i - n \bar x = n \bar x - n \bar x = 0 $$

- 离差的平方和等于$$ x_i$$的平方和减去$$ \bar x $$的平方和的n倍：

$$ \sum_{i=1}^n (x_i - \bar x)^2 = \sum_{i=1}^n x_i^2 - n(\bar x )^2 $$
