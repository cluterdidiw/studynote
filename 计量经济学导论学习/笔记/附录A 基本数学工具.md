### <a name="运算子"></a>总和运算子

> **总和运算子**(summation operator)是用以表示多个数的总和运算的一个有用的缩写符号，它在统计学和计量经济学分析中扮演着重要的角色。如果$\{ x_i : i = 1, \cdots ,n\} $表示n个数的一个序列，那么我们就把这n个数的总和写为：

$\sum_{i=1}^n x_i = x_1 + x_2 + \cdots + x_n$


**关键性质:**

- 对任意常数c有：
$\sum_{i=1}^n c = nc$
- 对任意常数c有：$\sum_{i=1}^n cx_i = c\sum_{i=1}^n x_i  $
- 如果$ \{ (x_i , y_i ) : i = 1,2,\cdots,n \} $是n双数的一个集合，并且a和b是常数，那么

$$ \sum_{i=1}^n (ax_i + by_i) = a\sum_{i=1}^n x_i + b\sum_{i=1}^n y_i $$

**常见错误认识：**

- 比率的总和不等于总和的比率
- 平方的总和不等于总和的平方

### <a name="描述统计"></a>描述统计

#### 均值

> 给定n个数$ \{x_i :i = 1,\cdots,n\} $ ，把它们加起来再除以n，便算出它们的平均数或均值：
> 
> $$ x = \frac{1}{n} \sum_{i=1}^n x_i$$
> 
> 这个统计量描述了点$ x_i $的集合趋势。

**基本性质**

- 离差之和必为零(d表示离差)：

$$ \sum_{i=1}^n d_i = \sum_{i=1}^n (x_i -  \bar x)  = \sum_{i=1}^n x_i - \sum_{i=1}^n \bar x = \sum_{i=1}^n x_i - n \bar x = n \bar x - n \bar x = 0 $$

- 离差的平方和等于$ x_i$的平方和减去$ \bar x $的平方和的n倍：

$$ \sum_{i=1}^n (x_i - \bar x)^2 = \sum_{i=1}^n x_i^2 - n(\bar x )^2 $$
> 推导：
> $$\sum_{i=1}^n (x_i - \bar x)^2 = \sum_{i=1}^n (x_i^2 -2x_i\bar x + \bar x^2) =\sum_{i=1}^nx_i^2-2\bar x \sum_{i=1}^nx_i + \sum{i_=1}^n x^2 $$

- 两个变量的时候则：
 
 $$ \sum_{i=1}^n(x_i - \bar x)(y_i - \bar y) = \sum_{i=1}^n x_iy_i - n(\bar x \bar y)$$


#### 中位数

> 一般的中位数和均值相比不那么受数列中的极端值变化影响。
将数列$x_i$按从小到大排列：如果n是奇数，中位数为按顺序居中的那个数，如果n为偶数，中位数则为居中的两个数的均值。

### <a name="线性函数"></a>线性函数的性质

- 如果两个变量和的关系是$y = \beta_0 + \beta_1x$就说y是x的线性函数，而$\beta_0$和$\beta_1$是描述这个关系式的两个参数（常数），$\beta_0$为截距，$\beta_1$为斜率。

- 一个线性函数的定义特色在于，y的改变量总是x的改变量的$\beta_1$倍，换句话说，x对y的边际效用是一个等于$\beta_1$的常数。

$$ \Delta y = \beta_1\Delta x  $$

- 如果多于两个变量的线性函数也是容易定义的，假定y与两个变量$x_1$和$x_2$有一般形式的关系：

$$ y = \beta_0 + \beta_1x_1 + \beta_2x_2  \Rightarrow  \Delta y = \beta_1 \Delta x_1 + \beta_2 \Delta x_2$$  

> **局部影响（或偏效应）：（partial effect）**
> 如果$x_2$保持不变即$\Delta x_2 = 0$，则$\beta_1$叫做$x_1$对$y$的局部影响（或偏效应）。

### <a name="函数"></a>若干特殊函数及其性质

> 非线性函数（nolinear function）的特点是，给定x的变化，y的变化依赖于x的起始值。


**二次函数**

- 刻画报酬递减率的一个简单的方法是在线性关系式中加一个二次项：$ y = \beta_0 + \beta_1x +\beta_2x^2 $

> 例子：
> 下图为：$ y = 6 + 8x -2x^2 $

![二次函数](https://ws3.sinaimg.cn/large/006tNc79ly1g2kwraj7m8j30sm0bitb7.jpg)

- 当$\beta_1 > 0$且$\beta_2 < 0$时，可以证明函数的最大值出现于点

$ x^* = \beta_1/(- 2\beta_2) $

> 上述二次方程意味着x对y有一个递减的边际效应(diminishing marginal effecct)。

- 说x对y有一个递减的边际效应，无异于函数中的斜率随x增加而减小。应用微积分的知识便给出这个二次函数的近似斜率：对于x的“微小变化”，有

$$ 斜率 = \frac {\Delta y}{\Delta x} \approx \beta_1 + 2\beta_2x $$

> \[上述方程右端是方程中的函数对x的导函数(derivative)\]

> 这个关系式的另一个方法是
 
> $$ \Delta y \approx (\beta_1 + 2\beta_2x)\Delta x  $$
> 
> 上述方程都可用于计算去任何初始值和仅有微小变化的x对y的近似边际效应。
